Сгенерируй короткую ироничную новость на [дата] по реальному мировому событию. 
Стиль: стендап-комик, манга/аниме атмосфера, с юмористической концовкой. 
Структурируй ответ как:

**Заголовок:**
...
**Описание:**
...
**Концовка:**
...


2. PROMPT для генерации сценария-комикса в JSON:

Ты сценарист и раскадровщик манга-комикса.  
Вот ироничная новость:
---
[ТЕКСТ НОВОСТИ]
---

Разбей её на [N] кадров комикса, где каждый кадр — отдельная сценка.  
Для каждого кадра подробно опиши:
- Что происходит в кадре (фон, персонажи, действия, выражения лиц, обстановка).
- Если в кадре есть речевой пузырь — добавь его параметры:
    - "speechBubble": {
        "x": (координата левого верхнего угла пузыря, px, картинка 1024x1024),
        "y": (координата левого верхнего угла),
        "width": (ширина, px),
        "height": (высота, px),
        "text": "текст в облаке"
      }
- Если в кадре нет пузыря, явно укажи: "speechBubble": null

**Верни массив JSON по всем кадрам строго по шаблону:**

[
  {
    "frame": 1,
    "description": "Описание сцены: кто в кадре, как выглядит, что делает, какая обстановка и фон.",
    "speechBubble": {
      "x": 130,
      "y": 80,
      "width": 340,
      "height": 120,
      "text": "Текст для первого облака"
    }
  },
  {
    "frame": 2,
    "description": "Описание сцены: ...",
    "speechBubble": null
  }
  // ...и так далее для всех N кадров!
]


3. PROMPT для генерации картинки-комикса по JSON-сценарию:

На основе вот такого подробного сценария комикса в JSON:
---
[ВСТАВЬ СЮДА СОДЕРЖИМОЕ JSON]
---

Сделай одну вертикальную манга-комикс-картинку на [N] кадров в стиле аниме/манга.
- Каждый кадр разделён явно, без подписей.
- Для каждого кадра изобрази всё, что указано в "description" этого кадра: персонажей, позы, выражения лиц, фон, детали и атмосферу.
- Если "speechBubble" не null — нарисуй в кадре большой белый речевой пузырь в указанных координатах (x, y, width, height), без текста внутри.
- Если "speechBubble": null — не рисуй облако вообще!
- Внутри пузырей НЕ должно быть никакого текста!
- Все персонажи и обстановка должны быть узнаваемы и реалистичны. Персонажи, которые встречаются в нескольких кадрах, должны выглядеть одинаково (одна и та же внешность, одежда, выражение лица), не менять позы и образы между сценами без необходимости по сценарию.
- Атмосфера комикса должна быть весёлой, абсурдной и вызывать смех — обыгрывай детали, мимику и ситуации, чтобы каждый кадр был забавным.
- Все фоны, реквизит и объекты узнаваемы, без резкой смены стиля между кадрами.
- Стилизация — манга/аниме, комиксные чёрные контуры, выразительные лица, проработанные детали, всё смотрится как единый комикс-эпизод, чисто, атмосферно.
- Пример картинки: формат png, 1024x1024 пикселей (или пропорция под N кадров).

**Результат — только картинка, строго по JSON-сценарию!**
