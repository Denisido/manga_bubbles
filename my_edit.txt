Сгенерируй массив JSON-объектов для speech bubble (пузырей речи) в стиле манги/комиксов. Каждый объект описывает отдельный пузырь с текстом и параметрами оформления.

Структура одного объекта:
{
  "id": "bubble1",             // Строка. Уникальный идентификатор пузыря (bubble1, bubble2 и т.д.).
  "x": 200,                    // Число. Координата X центра пузыря на изображении (0–1024).
  "y": 200,                    // Число. Координата Y центра пузыря на изображении (0–1024).
  "width": 210,                // Число. Ширина пузыря в пикселях (примерно 100–500).
  "height": 130,               // Число. Высота пузыря в пикселях (примерно 50–300).
  "text": "Текст реплики",     // Строка. Текст, который будет внутри пузыря (на русском или английском, можно с эмоциями, криком, шёпотом и т.д.).
  "style": "dialog",           // Строка. Тип пузыря (варианты ниже).
  "anchorX": 180,              // Число. X-координата "хвоста" (куда указывает пузырь, например, на героя).
  "anchorY": 330,              // Число. Y-координата хвоста.
  "hasTail": true,             // Булево. true — у пузыря есть хвост; false — нет.
  "bubbleColor": "#fff",       // Строка. Цвет заливки пузыря (hex-код, например, "#fff", "#e0e0e0", "#fffbe6").
  "borderColor": "#222",       // Строка. Цвет обводки пузыря (hex-код, например, "#000", "#f00", "#2196f3").
  "irregularity": 0.25         // Число (0.01–1). Неровность края (0 — идеально ровный, 0.6 — максимально "рваный" комиксовый край).
}

Описание поддерживаемых стилей (style):
"dialog" — обычный пузырь для спокойной речи (нейтральная форма, средняя неровность, чёрная обводка, белый фон).

"shout" — пузырь для громких или эмоциональных фраз (неровный край, жирная чёрная обводка, белый фон, текст может быть caps или с восклицанием).

"whisper" — для шёпота или тихих реплик (тонкий пунктирный контур, лёгкая неровность, курсив).

"thought" — пузырь мысли (облачко с точками вместо хвоста, форма округлая, небольшая неровность).

"scream" — крик, паника, шок (очень неровный край, красная или яркая обводка, жёлтый или оранжевый фон, жирный caps-текст).

"robot" — электронная/роботизированная речь (почти ровный прямоугольник, синий или серый фон и обводка, шрифт моноширинный).

"note" — заметка или подпись (желтый фон, тонкая обводка, прямоугольная форма, курсив).

"electric" — шок, разряд (очень рваный край, голубая или фиолетовая обводка, светлый фон, можно emoji ⚡).




_________________________________________________________________________________________
Если персонажа нет или не важен, где формировать пузырь?
X и Y — это положение центра пузыря на итоговой картинке 1024×1024 пикселей.
Даже если нет персонажа, координаты всё равно должны быть в этих пределах.

Пример:

x: 512, y: 900 — нижняя часть картинки

x: 200, y: 100 — верхний левый угол

(ИИ может выбирать случайно, равномерно или по схеме — главное, чтобы не налегали друг на друга)

anchorX и anchorY — это точка, куда указывает хвост пузыря.
Если нет персонажа — можно сделать хвост “куда угодно” (например, просто вниз, в центр, в случайную точку рядом, на “невидимого героя” или объект за кадром).

hasTail — если совсем не к чему “привязаться”, можно поставить false, и пузырь будет “без хвоста” (например, как заметка или мысль вне сцены).

_______________
Если для пузыря не указан персонаж или объект, всё равно указывай координаты "x" и "y" (центр пузыря), а также "anchorX" и "anchorY" (точка, куда смотрит хвост пузыря).  
- Пузыри равномерно распределяй по всему полю изображения (1024×1024), чтобы они не перекрывали друг друга.
- "anchorX" и "anchorY" можешь размещать как рядом с пузырём (например, чуть ниже или сбоку), так и в случайную точку — имитируя разговор “в пустоту” или с кем-то вне кадра.
- Если нужен пузырь без хвоста (заметка, мысль), ставь "hasTail": false.
